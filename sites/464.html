Meilisearch est un moteur de recherche open-source, qui dispose d’une solution Cloud.

  • OSS: https://github.com/meilisearch/meilisearch
  • Cloud: https://cloud.meilisearch.com/login

Nous sommes donc des amoureux de l’open-source et nous aurons à coeur de partager votre projet avec le reste de la communauté si celui-ci aboutit. On pourrait notamment vous aider à faire une mise en production long terme et parler du projet sur notre blog.

Le but du challenge : réaliser le site web le plus original intégrant les fonctionnalités de Meilisearch!

Pour vous inspirer et voir ce qui existe déjà, vous pouvez aller voir toutes nos démos.

 

Elements attendus

Réaliser un site web (front-end) dont la fonctionnalité centrale utilise une search bar, et les fonctionnalités associées (filtering, faceting…) de Meilisearch.

Biensur, ces fonctions doivent être pleinement intégré à l'application web du sujet de la nuit de l'info 2024.

Pas de techno imposée pour le site web.

La search bar en front-end doit communiquer avec une instance Meilisearch. Nous vous recommandons d’utiliser Meilisearch Cloud (https://cloud.meilisearch.com/login) pour set-up et customiser votre Meilisearch. Vous pouvez utiliser une instance de Meilisearch gratuitement pendant 14 jours.

La documentation de Meilisearch est disponible ici : https://www.meilisearch.com/docs

Nous disposons aussi de plusieurs intégrations pour faire communiquer votre front-end avec Meilisearch: https://github.com/meilisearch/integration-guides?tab=readme-ov-file#-sdks-for-meilisearch-api
